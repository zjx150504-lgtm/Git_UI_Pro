# 隐私说明

Git UI Pro 是本地桌面应用。默认情况下，软件只读取和操作用户主动添加的本地 Git 仓库。

## 本地数据

软件可能在本地保存：

1. 用户添加的项目路径。
2. 项目收藏、最近打开、排序等界面偏好。
3. 窗口布局、主题、面板尺寸等本地设置。

这些数据用于恢复用户工作区，不会主动上传到 Git UI Pro 作者或第三方服务器。

## Git 数据

软件会通过用户本机 `git` 命令读取：

1. 当前分支和仓库状态。
2. 提交历史、提交信息、作者信息和引用信息。
3. 工作区变更和 diff。
4. remote、ahead、behind 等同步状态。

当用户主动执行 fetch、pull、push 等远程操作时，网络请求由本机 Git 和仓库 remote 配置发起。

## 凭据

Git UI Pro 不保存 GitHub、Gitee、GitLab 等平台账号密码，不实现独立凭据系统。认证行为依赖用户本机 Git、SSH key、Git Credential Manager 或 credential helper。

## 日志和反馈

在反馈问题时，请先检查截图、日志和 Git 输出，避免公开：

1. 访问令牌和密码。
2. SSH 私钥。
3. 公司内部仓库地址。
4. 不希望公开的本地路径或提交内容。
